# XnetrouterV10.5 - Router Attack
**Fitur:**
- Icmp raw
- Icmp broadcast network
- Udp flood
- Ping combo

## Cara Instalasi
**Clone git**
```bash
git clone https://github.com/NekomonHub/XnetrouterV10.5
```
**Akses Directory**
```bash
cd XnetrouterV10.5
```
**Install modules**
```bash
npm install
```
## Cara Menjalankan
**Npm Start**
```bash
npm start
```
